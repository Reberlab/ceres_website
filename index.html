Lunar Grid in CERES


About the Task

For the Lunar coordinate finding task, the coordinate structure follows a very similar structure to the Military Grid Reference System (MGRS). This system has a 25km grid instead of the 100km grid that MGRS follows.

The current task uses point from the original Apollo 17 research stations (Landing Module, LEAM, Station 1, Station 3, Station 4, Station 5, Station 6, Station 7, Station 8, and Station 9) and all points are located in zone 27Q and 25km Grid DC.

The map used for this task were taken from Lunar Recognizance Orbiter Camera (LRO) and the map was resampled to 200cm. There are two demonstrations currently available, one where the map only has the lunar grid coordinate overlay and the other is with an annotated version. Both tasks use the same Apollo 17 research stations.
